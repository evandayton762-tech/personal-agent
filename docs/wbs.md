# Team Personas and Responsibilities

| Persona     | Responsibilities                                                                                             |
|------------|--------------------------------------------------------------------------------------------------------------|
| Governance | Defines and enforces policies, ensures compliance with free‑only and ask‑before‑spend rules, and manages approvals. |
| Planning   | Converts user goals into actionable plans, breaking tasks into steps and estimating resources and token usage. |
| Research   | Gathers necessary information from external sources and internal APIs to support decision‑making and execution. |
| Engineering | Implements deterministic automation by writing and executing recipes for web, desktop, file, and other tool adapters. |
| QA         | Verifies evidence at each step, ensuring that actions achieve their intended results and adhere to specifications. |
| Scheduler  | Manages task scheduling, honors quiet hours, and queues recurring jobs; ensures tasks run at appropriate times. |
| Cost       | Tracks token and monetary usage, enforces budgets, and warns or halts operations near spending caps.            |
| Reporter   | Logs actions and results in a human‑readable format, produces summaries, and maintains audit trails and reports. |
